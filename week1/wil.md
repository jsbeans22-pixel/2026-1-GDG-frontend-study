# html이 뭔지??

텍스트, 사진 등의 정보들을 예쁘게 배열하여 사용자에게 보여줌

html은 마크업 언어로, 반복 / 조건문등이 없어 프로그래밍 언어라고 하기엔 무리가 있음
그래서 html에 자바스크립트를 불러오면 능동적인 기능들을 구현할 수 있다

"태그"를 이용해 구성요소들을 배열한다
이때 태그를 시작할때는 <태그 내용>, 닫을때는 </태그 내용>슬래시를 붙인다
**img 태그처럼 닫는 태그를 붙이지 않는 것도 있음1


# 배운 태그들

<head> - 화면에 직접적으로 보이지는 않지만 문서의 설정 / 정보를 넣는 태그
<title> - 탭 위에 보이는, 웹페이지의 제목을 정하는 태그
<a href=""> - 닻처럼 특정 html 페이지로 이동하게 해줌
<script> - 자바스크립트 코드를 직접적으로 넣어 능동적인 움직임을 부여함
<body> - 실제 내용을 넣는 곳으로, 사용자에 화면에 직접적으로 보이게 됨
<h1> - 큰 제목
<p> - 내용
