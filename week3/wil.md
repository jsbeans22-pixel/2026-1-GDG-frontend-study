-cd로 현재 접근중인 디렉토리 변경 후 -npm start로 리액트를 켠다

리액트를 통해서 웹 애플리케이션의 구성 요소들을 전부 쪼개, 하나하나 작업이 가능함 - 굉장히 효율적인 협업이 가능

과거 여러개의 링크를 포함해 화면을 계속 전환하면서 웹 이용을 하던 때와는 다르게.. 리액트를 이용하면 화면전환 없이 작업 수행이 가능해짐(마치 하나의 앱처럼 작동)

일단 HTML, CSS, JavaScript를 잘 다루어야 이를 이용하는 리액트를 잘 다룰 수 있다



#배열
const name = []; 로 배열 생성(파이썬의 리스트와 유사)

#map 함수
const result = nums.map((x) => x * 2);
각 요소에 대해 함수를 실행 후.. 그 결과를 적용한 새로운 배열은 뱉어냄
파이썬의 map 함수와는 문법이 약간 다르고, 파이썬의 Lazy Evaluation(값을 바로 계산하지 않음)와는 다르게 호출하는 즉시 계산을 완료함
